# Credit Analysis App - Sistema de Análise de Crédito

Aplicação web em Next.js para avaliação de aprovação de crédito com base em regras de negócio.

## Regras de Negócio

1. Cliente deve ter 18 anos ou mais
2. Parcela (12x) não pode comprometer mais de 30% da renda mensal
3. Histórico de dívidas não pode estar negativado
4. Todas as condições devem ser atendidas para aprovação

## Tecnologias

- **Framework**: Next.js 14 (App Router)
- **Estilização**: Tailwind CSS
- **Persistência**: LocalStorage
- **Linguagem**: TypeScript

## Instalação

```bash
npm install
npm run dev
```

## Uso

Acesse `http://localhost:3000` após iniciar o servidor de desenvolvimento.
